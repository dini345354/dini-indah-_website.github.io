Kelas Pemrograman Web Dasar

Belajar membuat website dari nol hingga bisa membuat landing page sendiri.
Cocok untuk pemula yang ingin menguasai HTML, CSS, dan JavaScript.

Tentang Kelas

Kelas ini dirancang untuk membantu pemula memahami dasar-dasar pemrograman web mulai dari HTML, CSS, hingga JavaScript secara mudah dan terstruktur.

Belajar dari Nol

Materi disusun untuk pemula tanpa pengalaman coding, sehingga mudah dipahami dari dasar.

Materi Terstruktur

Pembelajaran disusun secara bertahap dari dasar hingga mampu membuat website sendiri.

Praktik Langsung

Setiap materi dilengkapi dengan praktik langsung agar lebih cepat memahami konsep.

Project Nyata

Peserta akan membuat project website sederhana sebagai hasil akhir pembelajaran.

Desain Menarik

Belajar membuat tampilan website yang menarik dan responsif menggunakan CSS.

Cocok untuk Pemula

Kelas ini sangat cocok untuk mahasiswa dan pelajar yang ingin mulai belajar coding.

Materi Pembelajaran

Pelajari dari dasar hingga mahir dengan 4 modul lengkap yang disusun secara sistematis

01

HTML Dasar

Belajar struktur dasar website, tag HTML, semantic element, dan form.

8 Pertemuan 18 Jam
02

CSS Dasar

Styling website, layout Flexbox & Grid, responsif dengan media query.

10 Pertemuan 22 Jam
03

JavaScript Dasar

Logika pemrograman, DOM manipulation, event handling, dan interaktivitas.

12 Pertemuan 26 Jam
04

Project Website

Membuat landing page profesional sebagai portofolio akhir.

6 Pertemuan 14 Jam
